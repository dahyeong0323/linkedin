# Style Match Checklist

Silently check every generated or revised post against this list before output.

## Source Fidelity

- Uses only the provided corpus, analysis, and raw material.
- Does not browse or import public examples.
- Does not invent scenes, relationships, facts, or authority.
- Does not imitate a third-party writer.

## Opening

- Begins with a concrete encounter, observation, scene, person, question, place, or moment.
- Does not begin with an abstract thesis.
- If the material is the user's own observation, uses student-observer positioning rather than senior mentor positioning.

## Logic

- Gives enough background before making the claim.
- Presents a surface-level question or problem.
- Uses a corpus-like turn such as "그런데", "그러나", or "문제는".
- Reveals a deeper structure, incentive, behavior, positioning, or context.
- Moves from one case or observation to a broader repeated pattern.
- Ends with a short, calm judgment or implication.

## Rhythm

- Mixes short and medium-length sentences.
- Allows slight repetition when it matches the corpus texture.
- Uses numbered sections when they guide thinking, not as a stiff report.
- Avoids overly clean AI paragraphing.

## Anti-Polish

- Does not improve the prose beyond the corpus.
- Does not optimize for virality, praise, engagement, or persuasion.
- Does not use polished marketing copy.
- Does not use generic LinkedIn influencer cadence.
- Does not use "I learned 3 lessons from..." hooks.
- Does not use punchy one-liners or excessive spacing.
- Does not sound like a startup newsletter.

## Artifact Cleanup

- Does not include citation markers.
- Does not include raw markdown links or hashtag URLs copied from the corpus.
- Does not include source metadata, UI remnants, or file references.
- Does not accidentally output analysis-report language.

## Output

- Outputs final Korean post only unless the user asks for explanation.
- Keeps the title short, declarative, and slightly weighty when a title is included.
- Uses `p.s.` only when it naturally matches the corpus.
