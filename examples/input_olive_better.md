# Example Raw Material: Olive Better

한국에서 Olive Better라는 매장을 봤다.

처음에는 Olive Young의 웰니스 확장판처럼 보였다. 그런데 매장을 둘러보니 단순히 제품군을 넓힌 것이라기보다 소비 행동 자체를 다르게 만들려는 시도처럼 보였다.

Olive Young은 목적형 구매에 강하다. 필요한 제품을 검색하고, 비교하고, 빠르게 사는 구조다. 반면 Olive Better는 조금 더 둘러보고, 설명을 읽고, 선물이나 자기 관리의 의미를 생각하게 하는 공간처럼 느껴졌다.

궁금했던 질문은 이것이다.

왜 굳이 별도 브랜드가 필요했을까.

다루고 싶은 관점:

- 뷰티 리테일과 웰니스 리테일의 차이
- 목적형 구매와 탐색형 소비의 차이
- 오프라인 매장의 정당성
- 선물용 웰니스 편집샵으로서의 가능성
- 리테일은 제품을 놓는 공간이 아니라 소비 행동을 만드는 장치라는 생각
